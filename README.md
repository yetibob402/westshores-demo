# West Shores Community Website - Demo

A professional recreation of the West Shores HOA community website, built to demonstrate seamless migration capabilities for MyLakeAccess.

## Features

- **Modern Next.js 15** with React 19 and TypeScript
- **Fully Responsive Design** - Works beautifully on all devices
- **Tailwind CSS** - Clean, professional styling
- **Complete Pages:**
  - Home (announcements, events, quick links)
  - About (community info, board of directors)
  - Amenities (8 community amenities with details)
  - Events (upcoming and past events calendar)
  - Documents (organized document library)
  - Contact (contact form + office information)

## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
npm run build
npm start
```

## Purpose

This is a demo site created to showcase how MyLakeAccess can seamlessly replicate and enhance existing HOA community websites. The design maintains familiarity while providing modern functionality and improved user experience.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Fully static-export ready

## Deployment

Can be deployed to:
- Vercel (recommended)
- Netlify
- Any static hosting platform

```bash
# Deploy to Vercel
vercel
```
